# 🎬 Netflix-Content-Strategy-Insights-Dashboard

## 📌 Project Overview

This project is an interactive **Netflix Content Analysis Dashboard** developed using **Power BI**. The dashboard provides valuable insights into Netflix's content library, allowing users to explore movies and TV shows across different countries, release years, and content categories.

The dashboard helps users understand content distribution, growth trends, content ratings, and geographical availability through interactive visualizations and filters.

---

## 🚀 Key Features

### 🔍 Interactive Filters
Users can dynamically filter the data based on:

- **Show Type** (Movie / TV Show)
- **Release Year**
- **Country**

These filters enable detailed exploration of Netflix's content catalog.

---

## 📊 Dashboard Insights

### 1. Total Content Overview
The dashboard displays key performance indicators (KPIs):

- **Total Shows:** 7,974
- **Movies:** 5,690
- **TV Shows:** 2,284

---

### 2. Content Distribution by Type
A donut chart visualizes the proportion of:

- Movies
- TV Shows

This helps identify the dominance of movies compared to television content on Netflix.

---

### 3. Global Content Distribution
A world map highlights the countries contributing content to Netflix.

**Insights:**
- Identifies major content-producing regions.
- Shows the global reach of Netflix's catalog.
- Helps compare content availability across countries.

---

### 4. Content Ratings Analysis
A bar chart displays the number of titles across different rating categories such as:

- TV-MA
- TV-14
- R
- PG-13
- PG
- TV-Y7
- TV-G
- NR
- Others

This provides an understanding of the target audience and maturity levels of Netflix content.

---

### 5. Content Growth Trend
A line chart illustrates how Netflix's content library has grown over time.

**Insights:**
- Rapid increase in content additions after 2010.
- Peak growth observed between 2017 and 2020.
- Useful for analyzing Netflix's expansion strategy.
